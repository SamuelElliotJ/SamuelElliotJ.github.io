StaffsVerse

TIGA Games Industry Awards 2022 'Education Initiative' finalist

This is a project I worked on as part of my universities 1UP initiative, which brought together students from a range of games courses and year groups to collaborate on a project. For this project I was part of the tech team and did visual scripting in Unreal Engine 5.

StaffsVerse is a virtual campus game which has the aim of helping new students used to the campus before they arrive. It will eventually be a 1:1 scale of the real Staffordshire University Campus.

Features I worked on:

-Player mini-map
-Level loading & unloading
-Emote menu

Images:

Learning Outcomes:

This project was my first time using Unreal Engine 5, so I first familiarised myself with the new layout and I then began working. My first task was a simple mini-map which is something I had done before so I finished it relatively quick. The next task was to create the emote wheel to the design specifications which had it as a radial menu, I hadn't done any radial UI before but with some research I quickly found out how to do it. My final task was to make a system to load and unload sub-levels, this took me the longest as I had never approached this before but I managed to get a basic system up and running.

Tools Used:

-Unreal Engine 5
